Abstract

In fast-evolving research fields such as 3D Gaussian Splatting (3DGS), World Models, Reinforcement Learning (RL), and Vision-Language-Action (VLA) models, the sheer volume of new papers and the fragmentation of information sources present a significant challenge for researchers. PaperMap introduces a paradigm shift from "passive paper reading" to "map-based active decision-making." By centering the organization of literature around entities of interest—such as specific companies, academic institutions, and key authors—PaperMap provides a stable, global perspective on the research landscape. This tool enables researchers to rapidly scan new contributions from major players in just 10 minutes a day, significantly reducing the cost of literature screening and facilitating more targeted deep reading.

Key Features

  • Entity-Centric Timeline: Track the evolution of research by viewing paper timelines filtered by specific companies, universities, or authors.
  • Abbreviation Visualization: Papers are displayed using concise abbreviations. Hovering over an entry reveals full metadata and AI-generated summaries.
  • Multi-Dimensional Views: Seamlessly switch between comprehensive paper lists, industry-focused views, and academia-focused views.
  • Intelligent Augmentation: Automatically fetches abstracts, generates concise summaries, and extracts institutional affiliations using AI.

Getting Started

To run PaperMap locally, clone the repository and execute the following commands:

git clone https://github.com/lihao2333/paper-map.git
cd paper-map
pip install panel pandas param pysqlite3
python dashboard.py

The local server will start at http://localhost:5006. For more detailed instructions, please refer to the README.
