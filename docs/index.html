PaperMap

论文全景地图

从「被动刷论文」到「基于地图的主动决策」

在线 Demo GitHub 仓库

核心价值

在 3DGS、WorldModel、RL、VLA 等方向，论文更新速度很快，信息来源又分散。PaperMap 以「关注对象」为核心，不再按信息来源被动接收，而是按你关注的公司、高校、作者主动组织论文，让你始终有一个稳定的全局视角。

每天早上约 10 分钟 即可快速扫描重点玩家的新工作，再决定是否深度阅读，持续降低论文筛选成本。

功能特性

  • 论文时间线：按公司 / 高校 / 作者查看论文时间线
  • 简称可视化：论文以简称展示，支持 hover 查看完整信息与 AI 摘要
  • 多维度视图：论文列表、关注的公司工作、关注的高校工作
  • 智能补全：自动获取摘要、生成总结、提取公司与高校信息

本地启动

克隆仓库后，执行以下命令启动：

pip install panel pandas param pysqlite3
python dashboard.py

服务器将在 http://localhost:5006 启动。更多说明请参阅 README。
